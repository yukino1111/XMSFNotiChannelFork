一键管理所有应用的**运营消息**通知通道，~~仅支持 MIUI~~仅在ColorOS 16和HyperOS 1.0环境测试通过，其他环境自测

## 测试环境

> 一加平板2Pro ColorOS 16.0.3  
> 一加Ace5至尊版 ColorOS 16.0.3  
> 红米K50 HyperOS 1.0.17  

## 使用方法

1. 在 Xposed 管理器 (如 LSPosed) 中激活模块
2. 作用域勾选 `系统框架(android)`
3. 重启设备
